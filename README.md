# My website

A very simple and probably too custom personal SPA website. Written in [Vue 3](https://vuejs.org/)
with [Vite](https://vite.dev/).

Available at: [visard.me](https://visard.me).

## Features

- Dark mode support
- Async image loading
- Blog posts written in Markdown

## Development

```bash
npm install
npm run dev
```
